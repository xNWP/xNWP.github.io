enum xtga::ERRORCODE : uint32
strong

a strongly typed enum containing error/status codes that can be raised during operation.

Enumerator
UNKNOWN 

An unknown error occured.

NONE 

Operation completed without error.

FILE_ERROR 

An error occured while opening/saving a file descriptor.

INDEX_OUT_OF_RANGE 

The requested index was out of range.

REDUNDANT_OPERATION 

The requested operation would be redundant.

OVERFLOW_DETECTED 

The requested operation causes an overflow.

INVALID_OPERATION 

The requested operation is invalid for the object.

INVALID_DEPTH 

The supplied image bit depth was invalid.

COLORMAP_TOO_LARGE 

The resulting color map wouldn't save space and thus was not returned.

CONTAINER_FULL 

The container is at max capacity and cannot have any new items added.
