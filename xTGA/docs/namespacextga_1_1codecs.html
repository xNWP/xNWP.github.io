Generates a Color Map from an input buffer.

Parameters
[in]inBuffThe input image buffer.
[out]outBuffThe encoded image buffer (Guaranteed 8-bits per pixel).
[out]colorBuffThe generated color map.
[in]lengthThe number of pixels the input buffer contains.
[in]depthThe bits per pixel of the input image (must be 16/24/32).
[out]SizeThe length of the exported color map.
[in]forceIf true a color map will be generated even if it won't perfectly represent the input data, this is done by taking the 256 most common values and then forcing each pixel to go to the closest common value (weighted).
[out]errorThe error/status code (can be nullptr).
Returns
bool True if the color map could be generated.
